| 指令                                          | 作用                        |
| ------------------------------------------- | ------------------------- |
| `cmake -B build`                            | 指定构建目录为 `build`           |
| `cmake -S . -B build`                       | 指定源码目录是当前目录，构建目录是 `build` |
| `cmake --build build`                       | 编译 `build` 目录里的项目         |
| `cmake --build build -j8`                   | 多线程编译，使用 8 个并行任务          |
| `cmake --install build`                     | 安装已构建的项目                  |
| `cmake -B build -DCMAKE_BUILD_TYPE=Release` | 配置 Release 构建             |
| `cmake -B build -DCMAKE_BUILD_TYPE=Debug`   | 配置 Debug 构建               |
| `cmake -B build -G Ninja`                   | 使用 Ninja 作为生成器            |
| `cmake --build build --target clean`        | 清理构建产物                    |
| `cmake --build build --target <target>`     | 只编译指定 target              |

最常见的完整流程是：
rm -rf build
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build -j8
cmake --install build
