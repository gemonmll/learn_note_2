你的疑惑非常好，而且你 **没记错** ——
**机体系相对于世界系确实在“动”**，所以有些向量在世界系看起来是变化的。

关键点在于：

### ❗机体系固定的向量 $(v_B)$

是 **在机体系中的表达**，它本身“不动”：
[
$\frac{d}{dt} v_B = 0$
]

但它变到世界系以后
[
$v_W = R(t)v_B$
]
就会随着 (R(t)) 变化而变化：

[
$\dot v_W = \dot R  v_B$
]
这才是我们观测到的“刚体在旋转”。

---

# 🚀 关键理解：

## ✔ 机体系固定的向量 **在机体系里不变**

## ✔ 但是在世界系中看起来是不断变化的

因为姿态矩阵 $(R(t))$ 在变。

---

# 🧠 举个最简单、能立刻理解的例子

假设机体系 x 轴方向向量是：

[
$e_x^B = (1, 0, 0)$
]

它是定死的（焊死在机体上）。

如果飞行器在绕 z 轴旋转，其世界系对应的向量是：

[
$e_x^W(t) = R(t) e_x^B$
]

随着时间变化，这个向量在世界系里绕着 z 轴旋转。
它当然有导数：

[
$\dot e_x^W = \omega \times e_x^W$
]

但 **这不是因为$ (e_x^B)$ 变化了**，而是因为：

* 世界系在看机体旋转
* $(R(t))$ 在变化
* 所以 $(v_W = Rv_B)$ 在变化

本质上：

[
$\dot e_x^W = \dot R e_x^B$
]

---

# ✔ 那为什么写成：

[
$\dot v_W = \dot R v_B + R \dot v_B$
]

这是链式法则（Leibniz rule）：

对乘积 (Rv_B) 求导：

[
$\frac{d}{dt}(Rv_B)
= \dot R v_B + R\dot v_B.$
]

但如果 (v_B) 是机体固定向量：

[
$\dot v_B = 0,$
]
所以：

[
$\boxed{\dot v_W = \dot R v_B}$
]

---

# ❗你记得“机体系相对世界系在动”是正确的！

只是：

* 我们写 (v_B) 时，指的是“此向量在机体系中的表达”
* 它不随时间改变（因为向量和机体一起旋转，是“固联”的）

但：

* 该向量在世界系中的值 (v_W(t)=R(t)v_B) 是变化的
* 变化完全来自 (R(t)) 的变化，即刚体的姿态变化

因此没有矛盾。

---

# 🌟 如果用四元数写，这一样成立：

[
$v_W = q \otimes v_B \otimes q^{-1}$
]

求导后也会得到：

[
$\dot v_W = [\omega_W]_\times v_W$
]

与旋转矩阵完全一致。
