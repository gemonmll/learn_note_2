打 tag 很简单，分两步：**在本地创建 tag** → **推送到远端**。建议用“带注释的 tag”（annotated tag），更规范。

---

## 1）先确认你要打 tag 的那个提交

通常就是当前 HEAD：

```bash
git status
git log --oneline -n 3
```

---

## 2）创建 tag（推荐：带注释）

```bash
git tag -a v1.0.0 -m "release v1.0.0"
```

> `v1.0.0` 换成你想要的版本号/名字。

---

## 3）推送 tag 到远端

只推这一个：

```bash
git push origin v1.0.0
```

或者一次推送所有本地 tag：

```bash
git push origin --tags
```

---

## 常用补充

### 给指定提交打 tag（不是当前 HEAD）

```bash
git tag -a v1.0.0 <commit_hash> -m "release v1.0.0"
```

### 查看 tag

```bash
git tag
git show v1.0.0
```

### 删除 tag

本地删除：

```bash
git tag -d v1.0.0
```

远端删除：

```bash
git push origin :refs/tags/v1.0.0
```
