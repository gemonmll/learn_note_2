## 快捷键一览

| 功能类别                          | 操作                   | 快捷键                    | 说明                                                                               |
| ----------------------------- | -------------------- | ---------------------- | -------------------------------------------------------------------------------- |
| **模式切换（MODE）**                | 进入选择模式（Select, 模型选择） | `Esc`                  | 按下 `Esc` 即可切换至选择模式。 ([Gazebo][1])                                                |
|                               | 进入旋转模式（Rotate）       | `r`                    | 按 `r` 切换至旋转模式，用于旋转模型。 ([Gazebo][1])                                              |
|                               | 进入平移模式（Translate）    | `t`                    | 按 `t` 切换至平移模式，用于移动模型。 ([Gazebo][1])                                              |
| **模型操作（MANIPULATING MODELS）** | 旋转模型                 | `r` + 点击模型 + 拖拽        | 快捷键 `r` 进入旋转模式，然后选中模型并使用鼠标拖动圆环控制旋转。 ([Gazebo][1])                                |
|                               | 移动模型                 | `t` + 点击模型 + 拖拽        | 快捷键 `t` 进入平移模式，然后选中模型并拖动箭头进行移动。 ([Gazebo][1])                                    |
|                               | 沿特定轴约束移动／旋转          | 同时按住 `x` / `y` / `z` 键 | 在平移或旋转时，按住 `x`、`y` 或 `z` 键可将操作约束在对应轴上。 ([Gazebo][1])                             |
|                               | 捕捉／跳跃移动或旋转           | 按住 `Ctrl` + 拖拽         | 在平移模式中按 `Ctrl` 可使模型按照固定增量（默认1 m）移动；在旋转模式中按 `Ctrl` 可按固定角度（默认45°）旋转。 ([Gazebo][1]) |
|                               | 相对于世界坐标轴对齐操作         | 按住 `Shift`             | 在平移或旋转时按住 `Shift` 表示切换为世界坐标系下的轴进行操作，而不是模型自身局部轴。 ([Gazebo][1])                    |
| **主窗口（MAIN WINDOW）**          | 保存窗口布局               | `Ctrl + s`             | 保存当前 GUI 的窗口／插件布局配置。 ([Gazebo][1])                                               |
|                               | “另存为”窗口布局            | `Ctrl + Shift + s`     | 将当前窗口布局另存为一个指定位置。 ([Gazebo][1])                                                  |
|                               | 加载窗口布局               | `Ctrl + o`             | 加载之前保存的窗口布局配置。 ([Gazebo][1])                                                     |
|                               | 退出 Gazebo            | `Ctrl + q`             | 关闭 Ignition Gazebo。 ([Gazebo][1])                                                |
|                               | 搜索插件／菜单项             | `/`                    | 在菜单中或插件界面使用 `/` 打开搜索菜单。 ([Gazebo][1])                                            |

## 实用说明与小技巧

* 在 **模型移动或旋转** 时，经常需要沿某一轴精确操作：例如将一个箱子沿 x 轴滑动一段距离。此时按住对应的轴键（例如 `x`）即可约束操作。
* 若你希望“对齐世界轴”而不是模型的本地轴，比如你旋转模型后，希望它再沿世界 Y 轴滑动，按住 `Shift` 会非常有用。
* 捕捉功能（`Ctrl` + 拖拽）可快速将模型移动／旋转至整洁的网格或角度，比如每隔 1 米或者45度，这在场景布置时非常高效。
* 保存和加载窗口布局（`Ctrl+s`, `Ctrl+Shift+s`, `Ctrl+o`）对多屏幕、多个插件面板或者长期项目非常重要：你可以自定义插件面板位置，保存为布局，以后快速恢复。
* 使用 `/` 打开搜索菜单可以快速定位插件或界面项，避免在菜单中层层查找。
* 当你在 GUI 操作中经常切换模式（选择 → 平移 → 旋转 → 选择 …）时，熟练使用 `t`, `r`, `Esc` 能极大提高效率。
