```cpp
bool ConditionNotifier::OpenOnly() {
  // get managed_shm_
  int shmid = shmget(key_, 0, 0644);
  if (shmid == -1) {
    AERROR << "get shm failed, error: " << strerror(errno);
    return false;
  }

  // attach managed_shm_
  managed_shm_ = shmat(shmid, nullptr, 0);
  if (managed_shm_ == reinterpret_cast<void*>(-1)) {
    AERROR << "attach shm failed, error: " << strerror(errno);
    return false;
  }

  // get indicator_
  indicator_ = reinterpret_cast<Indicator*>(managed_shm_);
  if (indicator_ == nullptr) {
    AERROR << "get indicator failed.";
    shmdt(managed_shm_);
    managed_shm_ = nullptr;
    return false;
  }

  ADEBUG << "open true.";
  return true;
}shmat 和shmget用法
```

你这段代码用的是 **System V 共享内存**。
可以把几个接口先记成一句话：

* `shmget`：**按 key 找到/创建一块共享内存，拿到 shmid**
* `shmat`：**把这块共享内存映射到当前进程地址空间，拿到可访问指针**
* `shmdt`：**把这块映射从当前进程解绑** ([man7.org][1])

## 1. 你这段代码在做什么

```cpp
int shmid = shmget(key_, 0, 0644);
```

这句是在说：

* 按 `key_` 去找一块**已经存在**的共享内存
* 因为 `shmflg` 里**没有** `IPC_CREAT`
* `size=0` 在这里表示“我不是要创建，只是获取已有段的 id”
  `shmget()` 既可以获取已有共享内存的标识符，也可以在带 `IPC_CREAT` 时创建新段。([man7.org][1])

然后：

```cpp
managed_shm_ = shmat(shmid, nullptr, 0);
```

这句是把 `shmid` 对应的共享内存挂到当前进程地址空间：

* `shmid`：共享内存 id
* `nullptr`：让内核自己选一个合适的、按页对齐的虚拟地址
* `0`：默认读写附加方式

成功后返回一个地址；失败时返回 `(void*) -1`。`shmat()` 的作用就是把指定共享内存段附加到调用进程的地址空间。([man7.org][2])

最后：

```cpp
indicator_ = reinterpret_cast<Indicator*>(managed_shm_);
```

这是把共享内存首地址当成 `Indicator` 结构体来用。
也就是说，这段代码默认约定：**这块共享内存的开头就是一个 `Indicator` 对象/布局**。

## 2. `shmget` 的典型用法

函数原型一般是：

```cpp
int shmget(key_t key, size_t size, int shmflg);
```

### 常见两种模式

### 只打开已有共享内存

```cpp
int shmid = shmget(key, 0, 0644);
```

含义：

* 不创建
* 只查找 key 对应的段
* 找不到就失败

你这段 `OpenOnly()` 就是这个意思。

### 创建或获取共享内存

```cpp
int shmid = shmget(key, sizeof(Indicator), IPC_CREAT | 0644);
```

含义：

* 如果已存在，就返回已有段 id
* 如果不存在，就创建至少 `sizeof(Indicator)` 大小的共享内存段
  POSIX 手册说明：当 key 不存在且设置了 `IPC_CREAT` 时，会创建一个至少为 `size` 的共享内存段。([man7.org][3])

### 想“只允许新建，已存在就报错”

```cpp
int shmid = shmget(key, sizeof(Indicator), IPC_CREAT | IPC_EXCL | 0644);
```

这样如果已经存在，会返回失败，错误常见是 `EEXIST`。([man7.org][3])

## 3. `shmat` 的典型用法

原型一般是：

```cpp
void* shmat(int shmid, const void* shmaddr, int shmflg);
```

### 最常见写法

```cpp
void* p = shmat(shmid, nullptr, 0);
```

含义：

* 让系统选地址
* 以默认方式附加
* 成功返回映射地址
* 失败返回 `(void*)-1` ([man7.org][2])

然后就能这样用：

```cpp
auto* indicator = static_cast<Indicator*>(p);
indicator->ready = true;
```

### 只读附加

如果你只想读，不想写，可以用只读标志：

```cpp
void* p = shmat(shmid, nullptr, SHM_RDONLY);
```

这样当前进程只能读这块共享内存。`shmat()` 支持通过 `shmflg` 控制附加方式。([man7.org][2])

## 4. `shmdt` 是干什么的

你代码里在失败路径有：

```cpp
shmdt(managed_shm_);
```

这个是把当前进程对这块共享内存的映射解除。
手册里说明，`shmdt()` 会把由 `shmat()` 返回地址对应的共享内存段从当前进程地址空间分离；如果某段已被标记删除，并且附加计数降到 0，它才会真正被删除。([man7.org][2])

所以：

* `shmdt` 只是**当前进程不再用**
* 不等于立刻销毁共享内存本身

真正控制删除通常还要 `shmctl(shmid, IPC_RMID, ...)`。([man7.org][4])

## 5. 你这段代码逐行解释

### 获取共享内存 id

```cpp
int shmid = shmget(key_, 0, 0644);
```

这里的语义是“只打开已有段”。
若 `key_` 没对应现成共享内存，返回 `-1`。

### 映射到当前进程

```cpp
managed_shm_ = shmat(shmid, nullptr, 0);
```

成功后 `managed_shm_` 指向当前进程中的一块虚拟地址，这块地址背后就是那段共享内存。

### 解释为你的业务结构

```cpp
indicator_ = reinterpret_cast<Indicator*>(managed_shm_);
```

于是后面就可以直接：

```cpp
indicator_->xxx = ...
```

这其实是在读写共享内存中的内容。

## 6. 这段代码里几个值得注意的点

### `indicator_ == nullptr` 这个判断基本没意义

```cpp
indicator_ = reinterpret_cast<Indicator*>(managed_shm_);
if (indicator_ == nullptr) { ... }
```

如果前面 `shmat()` 成功了，返回的不是 `(void*)-1`，那这里把它强转成 `Indicator*` 后通常不会变成 `nullptr`。
真正该检查的失败条件已经在前面这句里做了：

```cpp
if (managed_shm_ == reinterpret_cast<void*>(-1))
```

---

### `0644` 在这里是权限位，不是“open 风格权限”

`shmget` 第三个参数低 9 位是权限位，和文件权限的 rwx 概念类似，用来控制访问许可。POSIX 说明权限由 `shmflg` 的低 9 位指定。([man7.org][3])

---

### `size=0` 只适合“打开已有段”

如果你想创建共享内存，`size` 应该给出实际大小。
因为创建时需要指定“至少多大”的共享内存段。([man7.org][3])

## 7. 一个完整一点的最小示例

### 进程 A：创建并写入

```cpp
#include <sys/ipc.h>
#include <sys/shm.h>
#include <cstring>
#include <iostream>

struct Indicator {
  int value;
  bool ready;
};

int main() {
  key_t key = 0x12345;

  int shmid = shmget(key, sizeof(Indicator), IPC_CREAT | 0644);
  if (shmid == -1) {
    perror("shmget");
    return 1;
  }

  void* p = shmat(shmid, nullptr, 0);
  if (p == (void*)-1) {
    perror("shmat");
    return 1;
  }

  auto* ind = static_cast<Indicator*>(p);
  ind->value = 42;
  ind->ready = true;

  std::cout << "writer done\n";

  shmdt(p);
  return 0;
}
```

### 进程 B：只打开并读取

```cpp
#include <sys/ipc.h>
#include <sys/shm.h>
#include <iostream>

struct Indicator {
  int value;
  bool ready;
};

int main() {
  key_t key = 0x12345;

    perror("shmget");
    return 1;
  }

  void* p = shmat(shmid, nullptr, SHM_RDONLY);
  if (p == (void*)-1) {
    perror("shmat");
    return 1;

  shmdt(p);
  return 0;
}
```
